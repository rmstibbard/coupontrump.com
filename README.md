# Coupontrump.com

- An online education affiliate site written from scratch in HTML5, CSS (with Bootstrap), MySQL, PHP, jQuery, and AJAX. 
- Can be seen online at [http://coupontrump.com](http://www.coupontrump.com) but links are now disabled.
- Scraped Udemy course pages front-end HTML to find cheapest prices by applying coupon codes from around the web.
- Featured scheduled, automatic updating of coupons, scraping for new courses by author name, and many other automation features.
- Custom short-links for tracking.
- Advanced course search facility, which at the time was better than Udemy's search facility!

- Closed site due to changes in Udemy's affiliate rewards structure.
